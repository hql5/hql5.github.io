hugo搭建个人博客简要步骤

2019-05-17 times read

Contents

  1. 安装hugo
  2. hugo new site [path]
  3. 安装主题 themes.gohugo.io
  4. 本地启动hugo：hugo server –t [themes] –buildDrafts
  5. hugo new post/blog.md 新建博客
  6. 将个人博客部署到github服务器
    • 新建repository：[username].github.io
    • hugo –themes=m10c –baseUrl=https://[username].github.io/ –buildeDrafts
    • 当前文件夹下新建public文件夹
    • （public下）git init
    • git add .
    • git remote add origin https://github.com/lhq-lhq/lhq-lhq.github.io/
    • git commit -m []
    • git push -u origin master
